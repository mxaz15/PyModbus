Los archivos Sever y Cliente funcinan para escribir y leer 5 registros del tipo respectivamente.
Server escribe la fecha y hora desde el momento que se ejecuta.
Client consulta al server y escribe en pantalla el resultado.

La direccion por default del Server es 0.0.0.0 y el puerto 502 por lo tanto el Client debe apuntar a esa direccion y puerto.

MODO DE EJECUCION:
Abrir CMD en la carpeta de los archivos
Ejecutar python Server.py
Ejecutar python Client.py
